## 🏆 6. DIFFÉRENCIATEURS (MOAT)

### Ce que PERSONNE D'AUTRE ne fait

```
1. 📺 Asymétrie TV + téléphones
   - Un écran partagé au centre, les téléphones comme controllers
   - Force la présence physique, pas juste le scroll individuel
   - Concurrents: 100% mobile → pas de vision "salon"

2. 🌳 Arbre Neural navigable
   - Chaque conversation fait grandir un arbre visible
   - Votre relation devient quelque chose que vous pouvez VOIR et explorer
   - Concurrents: stats plates, pas de "monde" à naviguer

3. 🎧 DJ IA avec mémoire longitudinale
   - Il se souvient de tout ce que vous avez dit
   - Détecte les tensions, pose les bonnes questions au bon moment
   - Pas un simple quiz: un animateur intelligent

4. ⏰ Rituel synchrone 20h
   - Notification douce à 19h55, révélation à 20h00
   - Crée une habitude partagée, pas asynchrone
   - Concurrents: usage libre → pas de rituel, pas de rétention
```

### Ce qu'il faut ÉVITER de copier

```
❌ Les questions seules (commoditisable, IA gratuite peut le faire)
❌ Le profiling solo (faible rétention, pas de viralité)
❌ Le vocabulaire thérapeutique (risques juridiques, stigma)
❌ Les promesses de "réparation" (trop fort, pas crédible)
```

### Dashboard Minimal Viable (Supabase + Stripe)

```sql
-- Exemple de requêtes clés pour le dashboard

-- Revenu par source
SELECT 
  source, 
  COUNT(*) as purchases, 
  SUM(amount) as revenue
FROM purchases 
WHERE created_at > NOW() - INTERVAL '7 days'
GROUP BY source;

-- Conversion paywall
SELECT 
  COUNT(DISTINCT paywall_view) as views,
  COUNT(DISTINCT purchase) as purchases,
  COUNT(DISTINCT purchase) * 100.0 / COUNT(DISTINCT paywall_view) as conversion_rate
FROM user_events
WHERE event_type IN ('paywall_view', 'purchase');

-- Rétention couples
SELECT 
  DATE_TRUNC('week', first_login) as cohort_week,
  COUNT(DISTINCT user_id) as cohort_size,
  COUNT(DISTINCT CASE WHEN last_login > NOW() - INTERVAL '7 days' THEN user_id END) as retained,
  COUNT(DISTINCT CASE WHEN last_login > NOW() - INTERVAL '7 days' THEN user_id END) * 100.0 / COUNT(DISTINCT user_id) as retention_rate
FROM couple_users
GROUP BY DATE_TRUNC('week', first_login);
```
