# CAPTAIN BOND : LE MANIFESTE

## 1. L'Axiome "Anti-Tech"
L'écran n'est pas le jeu. L'écran n'est que l'étincelle. 
La technologie de Captain Bond a pour seul but de s'effacer le plus rapidement possible au profit de l'humain. Si les joueurs regardent leurs téléphones plus de 20% du temps, le jeu a échoué.

## 2. L'Économie de l'Intention
Dans une soirée classique, les humains perdent 80% de leur temps dans du "Small Talk" (Météo, Travail, Séries TV). 
Captain Bond est un accélérateur. Il brûle les étapes sociales pour forcer une connexion profonde et immédiate.

## 3. Le Droit à la Vulnérabilité
La vulnérabilité est perçue par le cerveau reptilien humain comme un danger absolu devant un public. 
Le rôle de l'architecture de l'application (Anonymat, Mode Feu de Camp, Safe Word) est d'offrir un bouclier numérique d'une sécurité totale pour désarmer les joueurs. L'application est un "videur bienveillant".

## 4. L'Asymétrie Numérique
Ce que les joueurs voient (La TV, l'écran de l'hôte) et ce qu'ils touchent (Leur téléphone) sont deux expériences totalement distinctes. 
- La Télévision est le "Feu de Camp" commun, solennel.
- Le Téléphone est le confessionnal privé.
Dans les modes ultra-intimes (Date Night), la technologie se restreint à un seul écran partagé posé à plat, forçant le contact visuel.

## 5. Le Silence est une Fonctionnalité
Le silence n'est pas un bug de rythme, c'est l'objectif. Le code inclut des temps de pause forcés (Pacing) pour obliger les joueurs à encaisser le poids des mots et à se regarder dans les yeux.
