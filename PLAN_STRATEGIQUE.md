## 1. Vision & Positionnement

### 1.1 Ce que Captain Bond EST

> **"Le moteur de connexions sociales qui transforme vos écrans en moments mémorables."**

Captain Bond n'est PAS :
- ❌ Une app de couple classique (type Paired, CoupleJoy)
- ❌ Un simple quiz de personnalité
- ❌ Un clone de Jackbox

Captain Bond EST :
- ✅ Un **écosystème de jeux sociaux** multi-ICP
- ✅ Un **moteur de viralité** (1 hôte = 5-8 invités)
- ✅ Un **cheval de Troie** : acquisition par le groupe, monétisation par le couple
- ✅ Une **plateforme d'expériences** propulsée par l'IA (Gemini)

### 1.2 Le "Hybrid Pivot" (Stratégie Gagnante)

**Ne pas pivoter à 180° vers une app couple. Conserver le moteur de viralité du jeu de groupe et l'utiliser comme Top-of-Funnel pour le mode couple.**

```
┌─────────────────────────────────────────────────────────────┐
│  TOP OF FUNNEL (Acquisition Virale)                         │
│  Jeu de groupe (amis, famille, collègues)                   │
│  → K-Factor > 1 (1 user = 5-8 nouveaux)                   │
│  → CAC quasi-nul                                            │
│  → Monetisation : Pass 24h (2.99€)                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  MIDDLE OF FUNNEL (Conversion)                              │
│  Révélation Profil Psychologique (Effet Barnum)             │
│  → "Vous avez 82% de compatibilité avec X"                  │
│  → CTA : "Découvrez votre Date Night"                       │
│  → Monetisation : Profile Pack (9.99€)                      │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  BOTTOM OF FUNNEL (Rétention MRR)                           │
│  Abonnement Couple (rituel quotidien)                       │
│  → LTV élevée (12-24 mois)                                  │
│  → Monetisation : 4.99€/mois                                │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 Pourquoi ce positionnement gagne

| Critère | Pivot 100% Couple | Hybrid Pivot (recommandé) |
|---------|-------------------|---------------------------|
| CAC | 15-30€ (ads) | ~0€ (viralité) |
| Dual onboarding | 60-80% de drop | 0% friction (QR code) |
| Marché | Océan rouge (Paired, Agape) | Océan bleu (gamified social) |
| Viralité | Faible | Élevée (K > 1) |
| ROI | Lent (18-24 mois) | Rapide (3-6 mois) |

## 2. Architecture ICP

### 2.1 Les 5 ICP de Captain Bond

| ICP | Nom | Âge | Mode(s) | Monetisation | LTV | Priorité |
|-----|-----|-----|---------|--------------|-----|----------|
| **ICP 1** | Hôte Social | 25-35 | Icebreaker, Spicy, Imposteur | Pass 24h (2.99€) | 8-15€ | 🔴 P0 |
| **ICP 2** | Couple | 28-45 | Date Night, Deep Connection | Abonnement (4.99€/m) | 60-120€/an | 🟢 P1 |
| **ICP 3** | Famille | Parents + ados | Family | Pass Famille (4.99€) | 12-20€/an | 🟡 P2 |
| **ICP 4** | Entreprise | DRH, Managers | Team Building | B2B (399€+) | 400-2000€/an | 🟠 P3 |
| **ICP 5** | Créateurs | 18-30 | Tous + partage | Freemium (ambassadeur) | 0€ (viralité) | 🔵 P0 |

### 2.2 Détail ICP 1 — Hôte Social (Patient Zero)

**Profil :**
- Jeune actif urbain, vie sociale active
- Organise des soirées (apéro, dîner, après-work)
- Veut être vu comme l'animateur cool

**Pain Point :**
> "Je veux briser la glace rapidement sans passer pour un weirdo avec des jeux ringards."

**Comportement d'achat :**
- Trigger : Soirée prévue dans les 48h
- Frein : "C'est juste pour une soirée"
- Prix psychologique : 2-5€ max

**Stratégie :**
- Canal : TikTok/Reels (réactions hilarantes)
- Message : "Le jeu qui a mis le feu hier soir 🔥"
- CTA : "3 cartes gratuites, puis 2.99€"

### 2.3 Détail ICP 2 — Couple (Mine d'Or MRR)

**Profil :**
- En couple depuis 1-10 ans, routine installée
- Cherche à raviver la flamme

**Pain Point :**
> "On tombe dans la routine, on ne se pose plus vraiment de questions."

**Comportement d'achat :**
- Trigger : Après avoir vu le "Profil de Compatibilité" en groupe
- Frein : "Est-ce juste un quiz de plus ?"
- Prix psychologique : 5-10€/mois

**Stratégie :**
- Canal : Upsell post-partie (CTA après révélation profil)
- Message : "Vous avez 82% de compatibilité. Découvrez votre Date Night."
- CTA : "Essai gratuit 7 jours, puis 4.99€/mois"

### 2.4 Détail ICP 3 — Famille

**Profil :**
- Parents 35-50 ans + enfants ados (13-18 ans)
- Familles recomposées ou nucléaires

**Pain Point :**
> "Les ados sont sur leurs téléphones, on ne communique plus."

**Stratégie :**
- Canal : Facebook Groups (parents), bouche-à-oreille
- Saisonnalité : Noël, vacances d'été
- Message : "Le jeu qui fait rire toute la famille (même les ados)"

### 2.5 Détail ICP 4 — Entreprise

**Profil :**
- DRH, Managers, Event Managers
- Entreprises 50-500 employés (PME/ETI)

**Pain Point :**
> "Les team buildings classiques sont chiants et coûteux."

**Stratégie :**
- Canal : LinkedIn, salons professionnels
- Message : "Team Building 2.0 : 100% digital, 0€ logistique"
- CTA : "399€ jusqu'à 20 personnes"

### 2.6 Détail ICP 5 — Créateurs (Amplificateur Viral)

**Profil :**
- Micro-influenceurs TikTok/Instagram (10k-100k followers)
- Cherchent du contenu engageant

**Stratégie :**
- Canal : DM Instagram/TikTok
- Message : "Le jeu qui génère des réactions hilarantes"
- CTA : "Code créateur : 50% de révenu"

## 4. Internationalisation SEO (EN/FR)

### 4.1 Choix Stratégique Validé

- **`/`** = Anglais (marché international, US-first)
- **`/fr/`** = Français (marché francophone)

### 4.2 Implémentation Technique

#### 4.2.1 Balises Hreflang (OBLIGATOIRE)

```tsx
// Dans app/layout.tsx ou via metadata
export async function generateMetadata({ params }) {
  const locale = params.locale || 'en';
  const baseUrl = 'https://captainbond.com';
  const path = locale === 'fr' ? '/fr/' : '/';

  return {
    alternates: {
      canonical: `${baseUrl}${path}`,
      languages: {
        'en': `${baseUrl}/`,
        'fr': `${baseUrl}/fr/`,
        'fr-FR': `${baseUrl}/fr/`,
        'fr-CA': `${baseUrl}/fr/`,
        'x-default': `${baseUrl}/`,
      },
    },
  };
}
```

> **Important :** Ne pas lister `en-US` et `en-GB` séparément vers la même URL. Google considère cela comme un signal faible ou erroné. Utiliser une seule entrée `'en'`.
>
> Pour les pages profondes (`/party/`, `/fr/soiree/`, etc.), générer les hreflang dynamiquement à partir de la table de correspondance URL EN/FR (§3.1).

#### 4.2.2 Sitemap.xml Multi-langue

Générer le sitemap dynamiquement à partir de la table de correspondance URL EN/FR (§3.1). Exemple :


#### 4.2.3 Redirections Géographiques

Ne jamais rediriger aveuglément. Utiliser un cookie de préférence de langue et un sélecteur visible.

```tsx
// middleware.ts
import { NextResponse } from 'next/server';
import type { NextRequest } from 'next/server';

export function middleware(request: NextRequest) {
  const country = request.geo?.country || 'US';
  const path = request.nextUrl.pathname;
  const preferredLocale = request.cookies.get('preferred_locale')?.value;

  // Si l'utilisateur a déjà choisi, respecter son choix
  if (preferredLocale) {
    return NextResponse.next();
  }

  // Redirection par défaut seulement pour la homepage racine
  if (path === '/' && country === 'FR') {
    const response = NextResponse.redirect(new URL('/fr/', request.url));
    response.cookies.set('preferred_locale', 'fr', { maxAge: 60 * 60 * 24 * 365 });
    return response;
  }

  return NextResponse.next();
}
```

> **Bonnes pratiques :**
> - Toujours afficher un sélecteur de langue dans le header/footer.
> - Ne jamais rediriger un expat ou un voyageur sans possibilité de revenir en EN.
> - Utiliser `NextResponse.redirect` (302) et non pas une 301, car la préférence peut changer.
> - Persister la préférence dans un cookie `preferred_locale` dès que l'utilisateur clique sur un sélecteur.

#### Leak #1 : Taux de conversion Freemium → Payant

**Actuel :** 1-3% (observé en MVP)
**Objectif :** 8-10%
**Solutions :**
- Préparer le paywall dès le tour 2 (pas de surprise au tour 4)

- Dynamic pricing (1.99€ en early-bird, 2.99€ ensuite)

**Actuel :** 5-10% d'upsell après fin de partie
**Objectif :** 15-25%
**Solutions :**
- Bundle Pass 24h + Dossier Classifié = 6.99€ au lieu de 7.98€
- Preview gratuite du profil individuel (axes floutés)

**Actuel :** 7.99€/mois et 39.99€/an (catalogue actuel)

- Maintenir 7.99€/mois (aligné marché EU)
- Pousser l'annual à 39.99€/an (3.33€/mois) comme choix par défaut
- Essai gratuit 7 jours

#### Leak #4 : B2B sous-exploité

**Actuel :** 0 prospection active
**Objectif :** 3-5 clients/mois, ACV moyen 9 600€/an (800€/mois)

- Offre d'essai gratuite

**Opportunité :** Permettre aux utilisateurs de donner un pourboire
**Revenu potentiel :** 20-50€/mois (1-2% des payeurs)
**Effort :** 1 jour de dev

## 7. Audit Éthique & Psychologique

### 7.1 Ce qui est validé scientifiquement

- ✅ Le jeu réduit le cortisol de 20-30% (Yeh et al., 2019)
- ✅ Les activités ludiques partagées augmentent l'ocytocine
- ✅ Les questions de vulnérabilité modérée renforcent les liens (Aron et al.)
- ✅ Le rituel du soir = "bid for connection" (Gottman Institute)
- ✅ Les questions ouvertes = base de la thérapie EFT

### 7.2 Points de vigilance psychologique

#### ⚠️ Point #1 : Le "Score de Résonance" peut créer de l'anxiété

**Problème :** Un score chiffré ("82% de résonance") crée :
- Comparaison sociale
- Anxiété de performance relationnelle
- Biais de confirmation

**Recommandation :**
- Remplacer par des qualitatifs ("Vous partagez une connexion rare sur les valeurs familiales")
- Ne JAMAIS afficher de "pourcentage de compatibilité" façon Tinder
- Disclaimer : "Ce score est indicatif et ne remplace pas une conversation"

#### ⚠️ Point #2 : L'effet Barnum doit être éthique

**Rappel :** L'effet Barnum = les gens croient qu'une description générique s'applique à eux.

**Recommandation :**
- ✅ Éthique si présenté comme "ludique"
- ❌ Non éthique si présenté comme "scientifique"
- Disclaimer : "Ce profil est généré à des fins de divertissement"
- Ne jamais utiliser de termes cliniques

#### ⚠️ Point #3 : La vulnérabilité forcée peut être contre-productive

**Recommandation :**
- Système de "progression de vulnérabilité" (questions légères → profondes)
- Safe Word (déjà implémenté ✅)
- Questions "skip" sans jugement

### 7.3 Perspective Thérapeute de Couple

#### 🔴 Problème #1 : Tu n'es pas un thérapeute

**Recommandation :**
- Disclaimer visible : "Captain Bond Couple est un outil ludique, pas un substitut à une thérapie"
- Page "Ressources" avec liens vers :
  - 3919 (violence conjugale France)
  - 09 72 39 40 50 (SOS Amitié)
  - Fil Santé Jeunes
- Support client formé à détecter les signaux de détresse

#### 🔴 Problème #2 : Le Safe Word est insuffisant

**Ce qui manque :**
- Détection de patterns à risque (réponses "je ne sais pas" systématiques)
- Message de soutien après activation ("C'est ok. Voici des ressources.")

#### 🔴 Problème #3 : Questions non validées cliniquement

**Recommandation :**
- Faire valider 100-200 questions par un psychologue clinicien
- Catégoriser par niveau de vulnérabilité (1-5)
- Ne pas poser niveau 4-5 avant la 5ème session

### 7.4 Lignes Rouges Éthiques

#### 🚫 Ligne Rouge #1 : Ne jamais prétendre être thérapeutique

**OK :**
- ✅ "Outil ludique pour renforcer les connexions"
- ✅ "Inspiré par les recherches en psychologie positive"
- ✅ "Créé avec contribution de psychologues"

**INTERDIT :**
- ❌ "Thérapie de couple digitale"
- ❌ "Guérissez votre relation"
- ❌ "Remplace une thérapie"

#### 🚫 Ligne Rouge #2 : Données sensibles

**Réalité RGPD :** Les réponses de couple sont des **données sensibles**.

**Obligations :**
- Consentement explicite (pas de case pré-cochée)
- Droit à l'oubli (suppression complète)
- Chiffrement de bout en bout
- Ne JAMAIS vendre ou partager

#### 🚫 Ligne Rouge #3 : Ne pas exploiter la vulnérabilité

**Recommandation :**
- Question onboarding : "Comment vous sentez-vous?" (1-5)
- Si ≤ 2 : proposer ressources avant de continuer
- Ne pas poser questions niveau 4-5 si détresse
