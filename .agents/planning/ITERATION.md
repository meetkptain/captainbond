# Itération — Amélioration Continue

## Cycle mensuel

```
Semaine 4 du mois
│
├─ Étape 1 : Analyser le scorecard (MESURE.md)
│
├─ Étape 2 : Identifier 3 meilleurs posts → les décomposer
│
├─ Étape 3 : Identifier 3 pires posts → comprendre pourquoi
│
└─ Étape 4 : Planifier le mois suivant (CALENDRIER.md)
```

## Analyse des meilleurs posts


## Analyse des pires posts

Pour chaque flop 3, noter :

```

Stage : 1 2 3 4
Format : Barnum / Question / Réaction
Pourquoi ça n'a PAS marché (hypothèse) :
  - Le hook était trop vague ?
  - La scène ne parlait pas ?
  - Le CTA était faible ?
  - Le timing était mauvais ?
  - Autre ?

Que changer : _________________
```

## Règles d'itération

1. **Ne JAMAIS changer le format et le message en même temps** — tu ne sauras pas ce qui a marché
2. **Si un format performe, le garder 2 semaines de plus** — même message, varier les scènes
3. **Si un message performe mais pas le format, changer le format** — garder la scène, essayer en carrousel ou question
4. **Si rien ne marche après 4 semaines, changer de plateforme** — aller sur Instagram Reels
5. **Les commentaires sont plus importants que les vues** — 100 coms sur 500 vues > 10 coms sur 10000 vues

## Évolution du contenu sur 3 mois

```
Mois 1 : Tester 32 scènes → identifier les 10 meilleures
Mois 2 : Doubler sur les 10 meilleures → 70% de contenu qui a déjà marché
Mois 3 : Ajouter UGC + réactions + contenu utilisateur
```

## Ajouter des variantes

Quand tu veux varier sans changer le message :

| Format | Description | Quand |
|--------|-------------|-------|
| Barnum 8s | Scène + résonance + direction | Base (70% du temps) |
| Question 8s | "Et toi, t'en es où ?" | Testing (15% du temps) |
| Carrousel 5 slides | La même scène en slides | Instagram (10% du temps) |
| Réaction | "Les 3 meilleures réponses de la semaine" | Engagement (5% du temps) |

## Pièges à éviter

- ❌ Changer de stage trop vite (attendre 2 mois minimum)
- ❌ Publier sans analyser (les données sont gratuites, utilise-les)
- ❌ Faire du contenu "pour tout le monde" (ça ne parle à personne)
- ❌ Ignorer les commentaires négatifs (ils disent ce qui cloche)
- ❌ Changer le format chaque semaine (l'algo a besoin de cohérence)
