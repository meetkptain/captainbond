# Calendrier Contenu — 12 Mois

## Principes

- 1 stage par 2 mois (8 semaines × 7 posts = 56 posts par stage)
- 4 stages = 8 mois de contenu sans répétition
- Mois 9-12 : rotation + UGC + réactions + itération

## Rotation des stages

```
2026
├── Septembre-Octobre   → Stage 1 : Nouveau Couple (1-3 ans)
├── Novembre-Décembre   → Stage 2 : Couple Coloc (5-10 ans)
├── Janvier-Février     → Stage 3 : Parents (jeunes enfants)
├── Mars-Avril          → Stage 4 : Post-Crise
└── Mai+                → Rotation + UGC + meilleurs moments
```

## Rythme mensuel

```
Semaine 1 : BATCH PRODUCTION (4h)
  └─ Produire 28 vidéos (4 semaines × 7 jours)
  └─ Écrire les descriptions
  └─ Programmer dans Buffer

Semaine 2-4 : EXÉCUTION (20 min/semaine)
  └─ Poster 1/jour
  └─ Répondre aux commentaires (10 min/jour)

Fin de mois : ANALYSE (30 min)
  └─ Lire MESURE.md
  └─ Identifier top 3 / flop 3
  └─ Ajuster pour le mois suivant
```

## Règle d'or

- Si un stage performe 2x mieux que les autres → prolonger de 1 mois
- Si un stage performe 2x moins → écourter, passer au suivant
- Toujours garder 1 mois d'avance dans la banque de contenu
