# Instagram Couple — Ritual Demo

Contenu "a faire ce soir" — rituels, exercices, questions.
Trigger : utilite immediate + faible friction.

## Format

- Plateforme : Instagram (Reel 30-60s)
- Duree : 30-45 secondes
- Structure : Hook besoin → Rituel → Comment faire → Resultat → CTA
- Musique : douce, chaleureuse, piano
- Alterner texte overlay + visuel appeasant

## Description copywritee + SEO

Chaque template genere automatiquement une description en 4 blocs :

```

Reprendre l'accroche de la video en mieux, plus profond.

BLOC 2 — CTA commentaire
→ Dis-nous en commentaire : {question simplifiee}
→ Tag ton/ta partenaire

BLOC 3 — CTA lien doux
3 questions gratuites → captainbond.com
{1 phrase de valeur, pas vendeuse}

BLOC 4 — 3 hashtags max
#couple #relation #captainbond
```

Structure exacte de la description :
```
{HOOK_REECRIT}

→ {CTA_COMMENTAIRE}
→ {CTA_TAG}

{LIEN_DOUX}

{3_HASHTAGS}
```

## Exemples concrets

### Exemple 1 : Le rituel du dimanche soir
```

DUREE : 20 minutes

→ Ce dimanche. 20 minutes. Essayez.
→ Tag ton/ta partenaire — faites-le ensemble

3 questions gratuites pour briser la routine → captainbond.com

#couple #relation #rituel"

DESCRIPTION : "Une question peut tout changer. Remplacez 'ca va ?' par ces 4 questions et regardez la difference.

→ Sauvegarde pour ce soir
→ Tag ton/ta partenaire — posez-vous les questions

3 questions gratuites → captainbond.com

#couple #relation #connexion"
```

## Notes

- Chaque rituel doit pouvoir etre fait CE SOIR (zero preparation)
- Les etapes doivent etre NUMEROTEES et claires
- Toujours inclure un "pourquoi ca marche" rapide
- Le CTA "Fais-le ce soir" convertit mieux que "sauvegarde"
- Montrer des visuels du rituel en action (meme reccreation)
