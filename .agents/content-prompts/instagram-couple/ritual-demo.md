# Instagram Couple — Ritual Demo

Contenu "a faire ce soir" — rituels, exercices, questions.
Trigger : utilite immediate + faible friction.

## Format

- Plateforme : Instagram (Reel 30-60s)
- Duree : 30-45 secondes
- Structure : Hook besoin → Rituel → Comment faire → Resultat → CTA
- Musique : douce, chaleureuse, piano
- Alterner texte overlay + visuel appeasant

## Template Prompt

```
Tu cree un contenu RITUEL pour @captainbond.couple (Instagram Couple).

Variables :
- {BESOIN} : le probleme que ce rituel resout
- {NOM_DU_RITUEL} : nom court et memorable
- {DUREE} : 5 min / 20 min / 30 min
- {ETAPES} : 3-5 etapes concretes
- {CTA} : "Fais-le ce soir"

Structure :
0-5s : {BESOIN} — "Vous vous sentez deconnectes ?"
5-15s : {NOM_DU_RITUEL} — presentation
15-35s : {ETAPES} — deroulement (texte overlay, etape par etape)
35-45s : {CTA}

Hashtags : #date night #couple #rituel #connexion #captainbond
```

## Exemples concrets

### Exemple 1 : Le rituel du dimanche soir
```
BESOIN : "Vous attendez lundi avec angoisse parce que vous n'avez pas passe de temps ensemble"
NOM : "Le Dimanche Check-in"
DUREE : 20 minutes
ETAPES : "1. Pas de telephones / 2. Chacun partage 3 moments forts de la semaine / 3. Chacun partage 1 difficulte / 4. Vous planifiez un moment pour la semaine prochaine / 5. Vous terminez par un câlin de 30 secondes"
CTA : "Ce dimanche. 20 minutes. Essayez."
```

### Exemple 2 : La question du soir
```
BESOIN : "Tu veux une conversation plus profonde que 'ca s'est passe comment aujourd'hui ?'"
NOM : "La Question du Soir"
DUREE : 10 minutes
ETAPES : "Au lieu de 'ca va ?', posez : 1. Quel moment t'a rendu.e heureux.se aujourd'hui ? / 2. Qu'est-ce qui t'a fatigue.e ? / 3. De quoi tu es fier.e ? / 4. Comment je peux t'aider ce soir ?"
CTA : "Teste ce soir et dis-moi en story"
```

### Exemple 3 : Le rituel Captain Bond Couple
```
BESOIN : "Vous voulez un moment de qualite mais vous ne savez pas quoi faire"
NOM : "Le Date Night Captain Bond"
DUREE : 30 minutes
ETAPES : "1. Ouvrez l'app Captain Bond / 2. Mode Date Night : ecran noir, questions profondes / 3. Chacun lit une question a voix haute / 4. Vous repondez sans vous interrompre / 5. Vous debriefez avec un câlin"
CTA : "3 questions gratuites. Lien en bio."
```

## Notes

- Chaque rituel doit pouvoir etre fait CE SOIR (zero preparation)
- Les etapes doivent etre NUMEROTEES et claires
- Toujours inclure un "pourquoi ca marche" rapide
- Le CTA "Fais-le ce soir" convertit mieux que "sauvegarde"
- Montrer des visuels du rituel en action (meme reccreation)
