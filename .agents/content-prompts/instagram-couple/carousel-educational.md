# Instagram Couple — Carousel Educational

Contenu evergreen a forte sauvegarde — listes, check-lists, diagnostics.
Trigger : utilite + autorite + "je garde pour plus tard".

## Format

- Plateforme : Instagram (Carrousel 5-10 slides)
- Format : images elegantes, typos serif, palette chaude
- Structure : Slide 1 (Hook) → Slides 2-9 (Contenu) → Slide 10 (CTA + Bio link)
- Pas de musique (carrousel = swipe, pas de son)

## Notes

- Les carrousels ont le meilleur taux de sauvegarde sur Instagram
- Toujours mettre plus de valeur que promis (underpromise, overdeliver)
- Les slides doivent etre beaux VISUELLEMENT (pas que du texte)
- Police serif pour les citations, sans-serif pour le corps
- Palette : ambre, rose, blanc casse, violet doux
- Derniere slide : toujours le lien en bio + incitation a commenter
- EVERGREEN : ces posts marchent encore 6 mois plus tard
