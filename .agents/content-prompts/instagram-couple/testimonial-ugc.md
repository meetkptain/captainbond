# Instagram Couple — Testimonial & UGC

Contenu "ca marche vraiment" — temoignages vrais de vrais couples.
Trigger : preuve sociale + identification ("si eux peuvent le faire, nous aussi").

## Format

- Plateforme : Instagram (Reel 30-60s)
- Duree : 30-60 secondes
- Structure : Temoignage → Contexte → Resultat → CTA
- Musique : douce, en fond, pas dominante
- Si possible : voix du vrai couple (pas de voix-off IA)

## Template Prompt

```
Tu cree un contenu TEMOIGNAGE pour @captainbond.couple (Instagram Couple).

Variables :
- {COUPLE} : profil type (10 ans ensemble, 1 enfant, etc.)
- {PROBLEME} : ce qui n'allait pas avant
- {DECOUVERTE} : comment ils ont trouve Captain Bond
- {RESULTAT} : ce que ca a change
- {CTA} : "essaie gratuit"

Structure :
0-5s : {HOOK} — la phrase choc du temoignage
5-20s : {COUPLE} + {PROBLEME} — contexte
20-40s : {DECOUVERTE} + {RESULTAT} — l'histoire
40-50s : {CTA} + logo

Hashtags : #temoignage #couple #amour #connexion #captainbond #ugc
```

## Exemples concrets

### Exemple 1 : Le couple qui allait mal
```
HOOK : "On etait a 2 doigts de se separer. Ce jeu nous a sauves."
COUPLE : "Maries 8 ans, 2 enfants"
PROBLEME : "On parlait plus que des gamins et du boulot. Je me sentais invisible."
DECOUVERTE : "Un soir, Captain Bond en mode Date Night. Questions profondes. Ecran noir, juste nous."
RESULTAT : "On a pleure. On a ri. On s'est retrouves."
CTA : "3 questions gratuites. Lien en bio."
```

### Exemple 2 : Le couple qui pensait tout savoir
```
HOOK : "Je pensais connaitre mon mari par coeur. J'avais tellement tort."
COUPLE : "Ensemble depuis 12 ans"
PROBLEME : "On etait en mode pilotage automatique"
DECOUVERTE : "Une question Captain Bond : 'Qu'est-ce que tu n'as jamais ose me demander ?'"
RESULTAT : "Il m'a demande quelque chose que j'attendait depuis 10 ans"
CTA : "Quelle question aimerais-tu qu'on te pose ?"
```

### Exemple 3 : Le couple qui ne sortait plus
```
HOOK : "On n'avait pas eu un vrai date night depuis 2 ans"
COUPLE : "Parents de jeunes enfants"
PROBLEME : "Sortir = baby-sitter = 80€ = on sort plus"
DECOUVERTE : "Captain Bond sur le canapé, apres le coucher des enfants"
RESULTAT : "Bien meilleur que n'importe quel restaurant. Et gratuit."
CTA : "Tague ton/ta partenaire pour lui suggerer"
```

## Notes

- Les vrais temoignages > recits inventes de tres loin
- Toujours demander la PERMISSION avant de reposter du UGC
- Si pas de UGC, recit realiste base sur des cas reels
- Ne jamais idealiser — les temoignages doivent inclure les difficultes
- Un bon temoignage = probleme + moment cle + transformation
