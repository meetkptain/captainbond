# TikTok Party — Identity Content

Contenu "c'est tellement moi" pour le Social Host.
Trigger : renforcement d'identite + vulnerabilite partagee.

## Format

- Plateforme : TikTok
- Duree : 15-30 secondes
- Structure : Hook (0-3s) → Corps (3-12s) → CTA (12-15s)
- Musique : son trending du moment ou voix-off
- Texte overlay TOUT LE LONG (lire sans son)

## Description copywritee + SEO

Chaque template genere automatiquement une description en 4 blocs :

```
BLOC 1 — Hook réécrit en phrase
Reprendre l'accroche de la video en mieux, plus profond.

BLOC 2 — CTA commentaire
→ Dis-nous en commentaire : {question simplifiee}
→ Tag l'ami/la personne concernee

BLOC 3 — CTA lien doux
3 questions gratuites → captainbond.com
{1 phrase de valeur, pas vendeuse}

BLOC 4 — 3 hashtags max
#motcle1 #motcle2 #motcle3
```

Structure exacte de la description :
```
{HOOK_REECRIT}

→ {CTA_COMMENTAIRE}
→ {CTA_TAG}

{LIEN_DOUX}

{3_HASHTAGS}
```

## Template Prompt

```
Tu cree un contenu IDENTITY pour @captainbond (TikTok Party).
ICP : Social Host (25-35), trigger identification.

Variables :
- {HOOK} : phrase d'accroche qui fait "c'est tellement moi"
- {LISTE_ITEMS} : 3-5 signes ou etapes
- {TON} : drole / vulnérable / relatable
- {CTA} : appel a action
- {DESCRIPTION} : generee automatiquement (hook reecrit + CTA com + lien doux + 3 hashtags)

{3_HASHTAGS}
```

## Exemples concrets

### Exemple 1 : Les signes
```
HOOK : "Tu sais que t'es l'host de ton groupe quand..."
ITEMS : "1. C'est toi qui lances le groupe WhatsApp / 2. Tu stresse 2h avant que tout le monde arrive / 3. Tu fais le tour pour verifier que tout le monde va bien / 4. Tu debrief le lendemain"
TON : drole mais vrai
CTA : "Tague l'host de TON groupe en commentaire"
DESCRIPTION : "Tu sais que t'es l'host de ton groupe quand ces 4 signes te parlent 😂

→ Tague L'ORGANISATEUR de ton groupe (il va se reconnaître)
→ Dis-nous en commentaire : quel signe ES-TU ?

3 questions gratuites pour ta prochaine soiree → captainbond.com

#hostlife #soiree #captainbond"
```

### Exemple 2 : L'anxiete d'avant-soiree
```
HOOK : "Moi 30 minutes avant ma soiree :"
ITEMS : "Je nettoie des trucs deja propres / Je change la playlist 4 fois / Je regarde l'heure toutes les 30 secondes / Je me demande pourquoi j'ai fait ca"
TON : vulnérable + drole
CTA : "Dites-moi que je suis pas le seul"
DESCRIPTION : "Les 4 signes que tu stresse AVANT que tes invites arrivent 😅

→ Avoue : t'as deja change la playlist 4 fois ?
→ Tag l'ami qui fait pareil

3 questions gratuites → captainbond.com

#hostlife #soiree #captainbond"
```

### Exemple 3 : Le debrief du lendemain
```
HOOK : "Le groupe WhatsApp le lendemain d'une soiree reussie :"
ITEMS : "Les photos volées / 'C'etait trop bien' x15 / 'On refait ca quand ?' / Le running gag qui commence"
TON : chaleureux, nostalgique
CTA : "Screenshot ton meilleur message du lendemain"
DESCRIPTION : "Le meilleur moment d'une soiree ? Le lendemain dans le groupe WhatsApp 📱

→ Screenshot ton message du lendemain et poste-le en story
→ Tag le groupe concerne

3 questions gratuites pour VOS prochaines soirees → captainbond.com

#debrief #soiree #captainbond"
```

## Notes

- Toujours overlay texte (90% vues sans son sur TikTok)
- Premier plan : visage ou reaction, pas un ecran noir
- Alterner humour leger et vulnerable vraie
- Les items impairs (3, 5) marchent mieux que pairs
