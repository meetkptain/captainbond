# TikTok Party — Humor / Sketch

Contenu "je taggue mes potes" — humour, sketches, archetypes de joueurs.
Trigger : reconnaissance + partage groupal.

## Format

- Plateforme : TikTok
- Duree : 15-30 secondes
- Structure : Hook sitcom (0-3s) → Sketches/Scenes (3-20s) → Chute (20-25s)
- Musique : son trending + bruitages
- Texte overlay : minimal (le visuel/acting porte)

## Template Prompt

```
Tu cree un contenu HUMOR pour @captainbond (TikTok Party).

Variables :
- {ARCHEYPE} : le type de joueur (le trop competitif, celui qui triche, le lent...)
- {SETUP} : la situation de depart
- {CHUTE} : le gag final
- {CTA} : appel a action drole

Structure :
0-3s : {SETUP} — la situation, le joueur
3-18s : Le jeu se deroule — moments cles accelerees
18-22s : {CHUTE} — le gag, la revelation
22-25s : {CTA} + logo

Hashtags : #soiree #fousrires #jeux #captainbond #partyfail
```

## Exemples concrets

### Exemple 1 : Le trop competitif
```
ARCHEYPE : Le joueur qui dit "je suis pas competitif" puis detruit tout le monde
SETUP : "Moi qui viens pour passer un bon moment tranquillement"
CHUTE : "2 minutes plus tard : JE VEUX LA REVANCHE"
CTA : "On est d'accord que c'est TOUJOURS celui qui dit pas competitif ?"
```

### Exemple 2 : Les 5 types d'invites a une soiree jeu
```
ARCHEYPE : Les archetypes du jeu de societe
SETUP : "Les 5 types de joueurs a chaque soiree Captain Bond :"
ITEMS : "1. Le strategique / 2. L'imposteur-ne / 3. Celle qui sait tout sur tout le monde / 4. Le lent / 5. Le tres serieux"
CHUTE : "Et toi t'es lequel ? (reponse honnete)"
CTA : "Tag ton pote le lent"
```

### Exemple 3 : Quand tu joues avec des gens bourres
```
ARCHEYPE : La partie apres 3 verres
SETUP : "Debut de soiree : tout le monde joue normalement"
CHUTE : "Fin de soiree : personne comprend plus les regles mais c'est le meilleur moment"
CTA : "Les meilleures parties commencent toujours comme ca"
```

### Exemple 4 : La question qui gene tout le monde
```
ARCHEYPE : Le moment Spicy
SETUP : *carte retournee* "Ma pire experience en date ?" (tout le monde se regarde)
CHUTE : Silence de 2 secondes puis tout le monde parle EN MEME TEMPS
CTA : "Quelle est la pire question qu'on vous ait posee ?"
```

## Notes

- L'humour TikTok est rapide (cuts toutes les 2-3 secondes)
- Pas de setup long — direct dans l'action
- Les archetypes marchent parce que les gens se reconnaissent ET reconnaissent leurs potes
- Toujours finir par un call to action qui invite au commentaire
- Si possible : montrer un extrait de VRAIE partie (UGC > reconstitution)
