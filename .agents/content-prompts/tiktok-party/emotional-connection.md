# TikTok Party — Emotional Connection

Contenu "ca me parle au coeur" — ce que l'hote ressent vraiment.
Trigger : vulnerabilite authentique + besoin d'etre vu.

## Format

- Plateforme : TikTok
- Duree : 30-60 secondes
- Structure : Hook calme (0-5s) → Recit (5-40s) → Revelation (40-50s) → CTA (50-55s)
- Musique : instrumentale douce, piano, lo-fi
- Texte overlay : present mais pas dominant (le recit porte)

## Template Prompt

```
Tu cree un contenu EMOTION pour @captainbond (TikTok Party).

Variables :
- {RECIT} : histoire personnelle liee au fait d'organiser des soirees
- {REVELATION} : le "plot twist" emotionnel
- {VALEUR} : ce que l'hote a appris
- {CTA} : reflet + appel

Structure :
0-5s : {HOOK} : question ouverte ou constat
5-35s : {RECIT} : deroulement de l'histoire (texte overlay + images douces)
35-45s : {REVELATION} : le moment cle, la prise de conscience
45-55s : {VALEUR} + {CTA}

Hashtags : #amitie #connexion #moments #coeur #captainbond
```

## Exemples concrets

### Exemple 1 : Ce que j'ai appris en organisant 50 soirees
```
HOOK : "J'ai organise 50 soirees en 3 ans. Voici ce que j'ai compris."
RECIT : "Au debut, je voulait que tout soit parfait. La deco, la musique, les invites. Puis un soir, tout a derape. Le jeu etait nul, la bouffe en retard, et pourtant..."
REVELATION : "...c'etait la meilleure soiree. Parce que les gens etaient la, ensemble."
VALEUR : "La perfection ne fait pas les souvenirs. Les gens les font."
CTA : "A tous les hotes qui stressent : vous faites deja assez bien."
```

### Exemple 2 : La soiree ou personne n'etait sur son telephone
```
HOOK : "Quand j'ai realise que mes amis s'aiment vraiment ?"
RECIT : "Un jeu un peu bete. Des questions. Des reponses. Et soudain tout le monde parlait, riait, SE REGARDAIT."
REVELATION : "Pendant 2 heures, personne n'a regarde son telephone. 2 HEURES."
VALEUR : "C'est pour ca qu'on organise des soirees. Pas pour les photos. Pour les regards."
CTA : "Ta meilleure soiree ? Raconte en commentaire."
```

## Notes

- Ce type de contenu est le PLUS partage en DM
- Il touche le coeur du besoin du Social Host : etre vu, etre important
- Toujours terminer sur une note d'espoir / de gratitude
- Les images doivent etre douces, chaudes, intimes
- Ne pas en abuser (1 pour 10 contenus humoristiques)
