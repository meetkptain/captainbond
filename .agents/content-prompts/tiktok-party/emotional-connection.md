# TikTok Party — Emotional Connection

Contenu "ca me parle au coeur" — ce que l'hote ressent vraiment.
Trigger : vulnerabilite authentique + besoin d'etre vu.

## Format

- Plateforme : TikTok
- Duree : 30-60 secondes
- Structure : Hook calme (0-5s) → Recit (5-40s) → Revelation (40-50s) → CTA (50-55s)
- Musique : instrumentale douce, piano, lo-fi
- Texte overlay : present mais pas dominant (le recit porte)

## Description copywritee + SEO

Chaque template genere automatiquement une description en 4 blocs :

```

Reprendre l'accroche de la video en mieux, plus profond.

BLOC 2 — CTA commentaire
→ Dis-nous en commentaire : {question simplifiee}
→ Tag l'ami/la personne concernee

BLOC 3 — CTA lien doux
3 questions gratuites → captainbond.com
{1 phrase de valeur, pas vendeuse}

BLOC 4 — 3 hashtags max
#motcle1 #motcle2 #motcle3
```

```
{HOOK_REECRIT}

→ {CTA_COMMENTAIRE}

{3_HASHTAGS}
```

## Notes

- Ce type de contenu est le PLUS partage en DM
- Il touche le coeur du besoin du Social Host : etre vu, etre important
- Toujours terminer sur une note d'espoir / de gratitude
- Les images doivent etre douces, chaudes, intimes
- Ne pas en abuser (1 pour 10 contenus humoristiques)
