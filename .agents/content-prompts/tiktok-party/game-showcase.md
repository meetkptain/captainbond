# TikTok Party — Game Showcase

Contenu "j'ai besoin de ce jeu pour ma prochaine soiree".
Trigger : utilite immediate + monnaie sociale (partager pour etre le heros de la soiree).

## Format

- Plateforme : TikTok
- Duree : 15-30 secondes
- Structure : Hook-probleme (0-3s) → Solution-jeu (3-10s) → Demo (10-20s) → CTA (20-25s)
- Musique : uptempo, beat marque, ou son trending
- Texte overlay permanent

## Template Prompt

```
Tu cree un contenu GAME SHOWCASE pour @captainbond (TikTok Party).

Variables :
- {PROBLEME} : le pain point (soiree qui tombe a l'eau, gens sur le telephone, awkward silence)
- {JEU} : le mode de jeu (Icebreaker / Spicy / Deep / Imposteur / Date Night)
- {NB_JOUEURS} : combien de personnes peuvent jouer
- {DEMO} : 5 secondes de ce qui se passe a l'ecran
- {CTA} : "3 cartes gratuites sur captainbond.com"

Structure :
0-3s : {PROBLEME} — texte overlay + visuel du probleme
3-10s : "Il y a un jeu pour ca" — transition vers le jeu
10-20s : {DEMO} — extrait du jeu en action
20-25s : {CTA} — ecran final + logo

Hashtags : #jeudesociete #partygame #soiree #captainbond
```

## Exemples concrets

### Exemple 1 : Icebreaker (briser la glace)
```
PROBLEME : "5 inconnus dans un salon. Qui parle en premier ?"
JEU : Icebreaker — questions legeres et absurdes
NB_JOUEURS : 3-8
DEMO : "A qui le tour ? Chacun lit une carte. Tout le monde vote."
CTA : "3 cartes gratuites. Lien en bio."
```

### Exemple 2 : Imposteur (soiree qui decollait pas)
```
PROBLEME : "La soiree etait morte... jusqu'a ce qu'on decouvre l'imposteur"
JEU : Imposteur — un joueur a une question secrete, les autres doivent trouver qui
NB_JOUEURS : 3-8
DEMO : "Tout le monde lit la question. Un seul bluffe. A vous de trouver."
CTA : "Tague le pote qui va adorer ca"
```

### Exemple 3 : Spicy (quand les amis deviennent competitifs)
```
PROBLEME : "Cette question a DIVISE la soiree en 2 camps"
JEU : Spicy — dilemmes A/B, le groupe choisit son camp
NB_JOUEURS : 2-8
DEMO : "Dilemme : embrasser un inconnu ou... L'autre option est PIRE"
CTA : "Reponds en story : A ou B ?"
```

### Exemple 4 : Deep Connection (version couple)
```
PROBLEME : "Vous pensez tout savoir l'un sur l'autre ?"
JEU : Deep Connection — questions vulnérables
NB_JOUEURS : 2-6
DEMO : "Ecran noir. Une question. A voix haute."
CTA : "Avec votre partenaire ce soir. Gratuit."
```

## Notes

- Toujours montrer le produit EN ACTION (capture d'ecran du jeu)
- Le probleme doit etre relatable en 1 seconde
- La transition probleme → solution doit etre claire
- Les questions du jeu elles-memes sont du contenu (les gens commentent leur reponse)
